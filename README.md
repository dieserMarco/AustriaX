# Feuerwehr Verwaltungssoftware

Eine schlanke Verwaltungssoftware für Feuerwehren zur Organisation von Mitgliedern, Einsätzen,
Geräten und Ausbildung.

## Ziele
- Zentrale Verwaltung von Stammdaten (Mitglieder, Fahrzeuge, Geräte).
- Schnelle Einsatzdokumentation und Nachverfolgung.
- Planung und Nachweis von Ausbildungen und Übungen.
- Transparente Auswertungen und Berichte.

## Geplanter Funktionsumfang
- Einsatzberichte schreiben (Formular) und Einsatzberichtsuche inkl. Berichtedruck.
- Einsatzkarte (fertiger Code wird eingebunden).
- Mitgliederverwaltung inklusive Kontaktliste und Mitgliederanmeldung (bedarf noch Klärung).
- Aktivitätsseite mit Übersicht über letzte Einsätze und Beteiligungen.
- Fahrzeugverwaltung (inkl. Wartungsfristen).
- Kursadministration für Ausbildungen und Übungen.
- Kassenverwaltung (wichtige Kernfunktion).
- Rettungskarten (nice to have).

## Prototypen
- `members.html` bietet einen ersten Entwurf für die Mitgliederverwaltung inkl. Kontaktliste,
  Statusfilter und Aktivitätsübersicht.
